# LSV Paper Tracker - User Guide

This is a quick guide for lab members using the tracker.

## What this site does

- Tracks how many papers each person reads per day.
- Shows activity as a GitHub-style heatmap per user.
- Shows a monthly leaderboard based on weekly goals.

## Daily use

### Option 1: Buttons

- `+1 today` adds one paper to today's count.
- `-1 today` subtracts one paper from today's count (never below 0).

### Option 2: Click heatmap cells

- You can edit only:
  - **today**
  - **yesterday**
- Click a cell to increment (`+1`).
- Shift+click a cell to decrement (`-1`).
- Older dates are locked and cannot be edited.

## Heatmap basics

- Each square is one date.
- Darker color means more papers on that day.
- Hover a square to see exact date and count.

## Leaderboard rules (monthly reset)

The leaderboard resets every month and scores by **week performance**:

- Weekly total `>= 5` papers: `+1` point
- Weekly total `3-4` papers: `0` points
- Weekly total `<= 2` papers: `-1` point

Important: for weeks that cross month boundaries, only days in the current month are counted.

## Tips

- Update daily to keep your streak realistic.
- Use click for quick updates while reviewing yesterday's log.
- If something looks wrong, refresh the page and try again.
